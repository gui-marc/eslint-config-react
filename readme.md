# Eslint Config React

A simple eslint config for react projects.

## Installation

```bash
npm install --save-dev @gui-marc/eslint-config-react
```

## Usage

Add the following to your `.eslintrc` file:

```json
{
  "extends": "@gui-marc/eslint-config-react"
}
```

And this to your `prettierrc` file:

```json
{
  "singleQuote": true,
  "trailingComma": "es5"
}
```
